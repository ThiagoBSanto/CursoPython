Atalhos Gerais:


COMENTAR LINHAS: Ctrl + /


Abrir ou fechar a paleta de comandos: Ctrl + Shift + P
Salvar o arquivo atual: Ctrl + S
Desfazer a última ação: Ctrl + Z
Refazer a última ação: Ctrl + Y
Recortar texto selecionado: Ctrl + X
Copiar texto selecionado: Ctrl + C
Colar texto do clipboard: Ctrl + V
Navegação no Código:

Ir para a definição: F12
Ir para a declaração: Ctrl + Click ou F12 em uma referência
Ir para o próximo erro ou aviso: F8
Ir para o último erro ou aviso: Shift + F8
Abrir um arquivo: Ctrl + P
Gerenciamento de Arquivos:

Criar um novo arquivo: Ctrl + N
Abrir um arquivo existente: Ctrl + O
Fechar a guia do editor ativa: Ctrl + F4
Alternar entre guias de editores: Ctrl + Tab
Terminal:

Abrir um novo terminal: Ctrl + Ç
Alternar entre terminais abertos: Ctrl + (acento grave)
Depuração:

Iniciar a depuração: F5
Parar a depuração: Shift + F5
Continuar a depuração: F5
Definir/remover ponto de interrupção: F9
Passo a passo: F10
Passo a passo em detalhes: F11
